## Key Features

- **Lock Task Mode (Pin):** prevents users from opening other apps or access the status bar. You can also enable the "Lock device when unpinning" feature to further enforce the security.
- **Protected Settings:** Changes to settings require biometrics or device credentials.
- **URL Filtering:** Supports regex-based blacklist and whitelist to control web access.
- **Export/Import:** settings & configurations

For all configurable settings, view the `Settings` section.
