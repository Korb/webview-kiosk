This application is currently under closed testing (google requires 12 active
testers for 14 days before it can be released publicly)

To test the app, you can **join the public google group below**:
- https://groups.google.com/g/webview-kiosk

Afterwords, you can install the app using:
- Web: https://play.google.com/apps/testing/com.nktnet.webview_kiosk
- Android: https://play.google.com/store/apps/details?id=com.nktnet.webview_kiosk

Once the application is stable and ready for open-testing, the source code
will be made available at https://github.com/nktnet1/webview-kiosk

In the meantime, if you have any questions, feel free to contact support at
**support@webviewkiosk.nktnet.uk**.
