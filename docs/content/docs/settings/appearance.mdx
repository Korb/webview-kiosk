import { Wrench } from 'lucide-react';

## Theme

Select one of

1. System (default)
1. Dark
1. Light

If either Dark or Light is selected, custom JavaScript will be injected
to override the `prefers-color-scheme` property of the WebView page.

## Address Bar Mode

Select one of

1. Hidden
1. Hidden when locked
1. Shown

With `Hidden when locked`, the address bar will be visible in the default view,
but disappears when the app is locked (pinned) using the floating toolbar <span className="inline-flex">(<Wrench className="-rotate-90"/>)</span>.

## Blocked Message

The message to display when a URL matches the blacklist regex filter rule.
